# Skyport Panel

Skyport is an game server panel built for the modern age. It's a drop in replacement for Pterodactyl, Pelican and others - with loads of nice new features plus an entirely rebuilt (from the ground up!) codebase.

It's easy to get started. One command to install the panel. One command to install the daemon. It can't get any easier than that.

- Documentation: [skyport.sh](https://skyport.sh)
- Support: [Join our Discord](https://discord.gg/PUuezaSqkY)
